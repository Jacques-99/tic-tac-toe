# Tic-Tac-Toe

This is a basic tic-tac-toe game.

## Installation

There are 2 methods to install the required packages for the project:

### 1. Requirements.txt:

```
pip install -r requirements.txt
```
### 2. Poetry:

```
pip install poetry
poetry shell
poetry install
```

## Run the application:

Simply run the following command:
```
python main.py
```
